# Development workflow.

Features will be developed using Spec Driven Development.

The requirements specification will be written to `requirements.md`.

The technical specifications saved to `specs.md`


`specs.md` should be broken down into tasks, when prompted to create an implementation plan, it should be saved to tasks/<feature_name>/implementation_plan.md

When completing an implementation plan write a summary of design descisions, challenges and tradeoffs to tasks/<feature_name>/implementation_summary.md
